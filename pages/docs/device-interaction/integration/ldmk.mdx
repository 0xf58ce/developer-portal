# The Ledger Device Management Kit

The Ledger Device Management Kit (LDMK) provides a Typescript library to easily handle Ledger devices:
- Device enumeration, identification and connection management
- Device actions, with full status and error report
- In the future: Firmware and Applications installation/update.

As more documentation is being created to help you use the LDMK, you can read follow [the official repository ReadMe](https://github.com/LedgerHQ/device-sdk-ts) for more information.
